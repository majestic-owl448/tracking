# freeCodeCamp Contribution Activity Tracker

This repository tracks weekly contribution activity for freeCodeCamp projects.

## GitHub Pages Setup

To publish this as a GitHub Pages site:

1. Go to your repository on GitHub
2. Navigate to **Settings** > **Pages**
3. Under "Source", select **Deploy from a branch**
4. Select **main** branch and **/ (root)** folder
5. Click **Save**

Your site will be available at: `https://<your-username>.github.io/<repo-name>/`

## File Structure

- `activity-YYYY-MM-DD.md` - Weekly activity logs
- `index.md` - Main page listing all activity logs
- `_config.yml` - Jekyll configuration
- `_layouts/` - Custom layouts (optional)

## Adding New Weekly Logs

Activity files follow the naming convention: `activity-YYYY-MM-DD.md` where the date represents the Monday of that week.

## Theme

This site uses the GitHub Pages default theme (Minima). You can customize it by:
- Editing `_config.yml` to change theme settings
- Adding custom layouts in `_layouts/`
- Adding custom CSS in `assets/css/`
